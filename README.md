# KIMIS: Intelligent Knowledge Management System

## Overview
KIMIS is a web-based Knowledge Management System featuring a Visual Workflow Engine and a Custom AI Integration module. It allows users to create, manage, and retrieve knowledge articles through standard search and NLP keyword matching.

## Features
* **Core:** JWT Authentication, Role-Based Access Control (RBAC)
* **Knowledge Base:** Article CRUD with tags and categorization
* **Visual Workflow:** Node-based workflow creation and mapping
* **AI Retrieval:** NLP keyword matching and step-by-step response engine
